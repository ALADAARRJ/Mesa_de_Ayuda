REQUISITOS:
	1.- Tener instalado XAMPP
	2.- Tener instalado Visual Studio Code

INSTRUCCIONES:
	1.- Pasar todo el proyecto (Carpeta "mesaAyuda") a la direccion "C:\xampp\htdocs".
	2.- Pasar la base de datos (Carpeta "mesa_ayuda") a la direccion "C:\xampp\mysql\data.
	3.- Encender en XAMPP los servicios de Apache y MySQL.
	4.- Ir al navegador a la ruta de localhost y seleccionar el proyecto mesaAyuda
	5.- Ingresar con las credenciales de la base de datos segun las funciones que se queiran probar.
